函式

Posted on 2024-06-13 In JavaScript
JavaScript 函式的定義和使用方法，包括基本概念和進階應用。瞭解函式的運作原理對於掌握 JavaScript 至關重要，因為函式在這門語言中扮演著核心角色。通過實踐和運用上述概念，能夠更好地開發和維護 JavaScript 應用程式。

定義函式

函式的定義由以下幾部分組成：

  1. 函式名稱
  2. 參數列表，用括號 () 包圍，參數之間用逗號分隔
  3. 函式主體，用大括號 {} 包圍，包含一系列的 JavaScript 語句

函式 square 有一個參數 number，且僅有一行程式碼，返回 number 自乘的結果。

原始參數（例如數字）作為值傳遞給函式，如果函式改變了這個參數的值，不會影響函式外部的原始變數。然而，若傳遞一個物件（如陣列或自定義物件）作為參數，函式改變了這個物件的屬性，改變會影響到函式外部的物件，因為傳遞的是物件的位址。

請注意，重新賦值參數並不會影響函式外部，因為只是改變了參數的值而不是物件的屬性。

預定義函式

JavaScript 提供多個頂級預定義函式，如 eval、isFinite、isNaN、parseInt 和 parseFloat。這些函式對於不同的操作非常有用。
