JavaScript JSON

Posted on 2024-06-10 Edited on 2024-06-11 In JavaScript
JavaScript Object Notation（JSON）是一種用於將結構化資料呈現為JavaScript物件的標準格式。在網頁開發中，JSON被廣泛用於資料呈現和傳輸，對於理解JavaScript與JSON的關係至關重要。本文將深入探討JavaScript物件與JSON格式，以及如何運用它們建立自定義的JSON物件。

JSON（JavaScript Object Notation）是一種標準的資料格式，用於表示結構化資料，常用於網站的資料傳輸和呈現。JSON以JavaScript物件的形式存在，並且可以獨立於JavaScript使用。JSON的應用範圍廣泛，從簡單的資料到複雜的資料結構都可以使用。

物件與文字交互轉換

我們接收到文字字串格式的 JSON 資料，需將之轉換回物件。且當我們要以某種訊息傳送 JSON 資料時，也必須將之轉換為字串才能正確運作。內建的 JSON 物件很早就新增到瀏覽器之中，且包含下列 2 種函式：

  • parse()：接收文字字串形式的 JSON 物件作為參數，並回傳對應的物件。
  • stringify()：接收 JSON 物件作為參數，並回傳對等的文字字串形式。

其他注意事項

  • JSON是一種純粹的資料格式，僅具有屬性，不含函式。
  • JSON的屬性值必須使用雙引號。
  • JSON的語法必須嚴格符合規定，否則會導致資料無法正常解析。
  • JSON的應用範圍廣泛，不僅限於物件或陣列。
